บทคัดย่อ

  ปัจจุบันสุขภาพดีเป็นสิ่งที่ทุกคนต้องการ แต่เพราะว่าไม่ค่อยมีเวลาและขาดความรู้ความเข้าใจในการออกกำลังกาย จึงไม่สามารถออกกำลังหรือดูแลสุขภาพให้เป็นตามที่ต้องการได้ จึงทำให้สุขภาพร่างกายไม่แข็งแรง เหนื่อยง่าย ดังนั้นคณะผู้จัดทำ จึงจัดทำโครงงานนี้ขึ้นเป็น Application ที่มีวัตถุประสงค์เพื่อแนะนำแนวทางการออกกำลังกาย โดยโปรแกรมของคณะผู้จัดทำจะแสดงข้อมูลต่างๆ เบื้องต้นโดยอ้างอิงจากเป้าหมายของผู้ใช้ เช่น วิธีการออกกำลังกาย และโภชนาการที่เหมาะสมต่อบุคคลนั้นๆ ซึ่งสามารถทำให้ผู้ใช้สามารถแบ่งเวลาในการดูแลสุขภาพตนเองได้อย่างเหมาะสม



การทำงานของโปรแกรม

  โปรแกรมจะรับ Input ที่เราใส่เข้าไป จากนั้นจะนำไปคำนวณตามสูตรที่กำหนดไว้ เมื่อคำนวณเสร็จแล้ว Output จะออกมาเป็นค่าBMI และคำแนะนำในการออกกำลังกาย
